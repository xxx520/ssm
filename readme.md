### 项目简介 
- [x] 使用 SpringBoot 2.1.2.RELEASE
- [x] JPA + Mybatis-Plus 任意切换
- [x] aop日志记录方式
- [x] 代码生成 
- [x] 动态权限管理 菜单按钮权限+数据权限配置
- [x] 后台提供分布式限流、同步锁、验证码等工具类 前端提供空白Vue模版


### 项目结构 
- base 
- common util,aop,contants
- config 拦截器，配置
- generator 代码生成器
- modules	base（系统相关）api（业务相关）
